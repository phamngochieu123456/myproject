Three trees of Git:
- Working Tree: chứa những tập tin hiện tại được theo dõi bởi git và các tập tin chưa được theo dõi
- Staging
- Commit

Từ khóa học:
- Working Tree: chứa những tập tin hiện tại được theo dõi bởi git và các tập tin chưa được theo dõi
- Git directory: như là một cơ sở dữ liệu chứa những thay đổi của file được theo dõi bởi git
- Staging Area: một file được duy trì bởi git chứa thông tin về file nào và thay đổi nào sẽ đi vào lần commit tới
    + Sử dụng git status để xem Working Tree và các thay đổi đang chờ xử lý
    + Git chú ý bất cứ khi nào thay đổi tập tin nhưng không lưu cho đến khi file được thêm vào Staging Area( sử dụng git add, khi sử dụng
    git add, các file thay đổi trở thành staged files)

- Để commit vào git directory, sử dụng lệnh git commit  

- Sau khi add index1.html với nội dung sai, thực hiện: 
git reset head index1.html
git checkout index1.html
